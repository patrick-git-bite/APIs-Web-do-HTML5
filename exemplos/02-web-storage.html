💾 Web Storage API

Armazena dados no navegador do cliente. localStorage persiste mesmo após fechar o navegador. sessionStorage dura apenas enquanto a aba estiver aberta.

🟢 localStorage (persistente)

Demo prática: Salvar seu nome

Digite seu nome abaixo. Ele será salvo no localStorage. Feche a aba e reabra — seu nome continuará aqui!

🟠 sessionStorage (temporário)

⚠️ Feche esta aba e reabra a página — os dados do sessionStorage terão desaparecido.

Código-fonte

// === localStorage (persistente) ===
localStorage.setItem("nome", "Patrick");
localStorage.getItem("nome");       // → "Patrick"
localStorage.removeItem("nome");
localStorage.clear();                // remove tudo

// === sessionStorage (temporário) ===
sessionStorage.setItem("etapa", "3");
sessionStorage.getItem("etapa");     // → "3"
sessionStorage.removeItem("etapa");
sessionStorage.clear();
