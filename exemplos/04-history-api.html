🔄 History API

Manipula o histórico de navegação sem recarregar a página. Essencial para SPAs (Single Page Applications).

Simulação de SPA — Navegação sem recarregar

🔒

📜 Log de Navegação

Cada ação registrada aqui mostra o que a History API fez nos bastidores.

[início] Página carregada

Código-fonte

// Adiciona nova entrada no histórico (URL muda, página NÃO recarrega)
history.pushState(
  { pagina: "contato" },  // objeto de estado
  "",                      // título (ignorado pela maioria)
  "/contato"               // nova URL
);

// Substitui entrada atual (sem criar nova)
history.replaceState(
  { pagina: "sobre" },
  "",
  "/sobre"
);

// Detecta navegação com voltar/avançar
window.addEventListener("popstate", (event) => {
  console.log("Estado:", event.state);
  // Atualizar a interface com base no estado
});

// Navegação manual
history.back();     // Voltar
history.forward();  // Avançar
history.go(-2);     // Voltar 2 páginas
